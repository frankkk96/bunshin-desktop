在 Bunshin 里接入模型

Bunshin 通过 Claude Code 运行,任何提供 Anthropic 兼容接口 的模型都能用。下面是几家常见国产模型的配置方式。

通用步骤

  1. 在下面对应 Provider 的控制台创建一个 API Key。
  2. 在 Bunshin 里 新建 Agent,把该 Provider 的 Base URL、API Key、Model 三项填进去。
  3. 开始会话即可。
重要 · 补充三个环境变量
Claude Code 内部会按 opus / sonnet / haiku 三档请求模型,只填一个 Model 时,子代理/后台调用会回退到真正的 Claude 模型名,第三方接口不认。请在 Agent 的「高级配置 → 环境变量」里把这三项都指向该 Provider 的模型 id:
变量ANTHROPIC_DEFAULT_OPUS_MODEL = <模型id> ANTHROPIC_DEFAULT_SONNET_MODEL = <模型id> ANTHROPIC_DEFAULT_HAIKU_MODEL = <模型id>
模型 id 更新较快,以各家控制台「模型列表」为准;下方给出的是撰写时的推荐编码模型。海外/国内为不同站点,Key 不通用,请按所在区域选择对应的 Base URL 与控制台。
